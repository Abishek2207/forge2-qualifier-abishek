# Evidence Mapping

This file maps each requirement of the Forge 2 Qualifier to its repository proof.

* **OpenClaw loop** → `agent-log.md` (Contains task logs, showing OpenClaw task loop, execution, and status reporting)
* **Hermes memory** → `memory/hermes_memory.json` (Proves persistent memory functionality for Hermes)
* **Skill** → `skills/` (Folder containing predefined agent skills)
* **Autonomous run** → `outputs/autonomous-run.txt` (Proof of the script triggering the agent check autonomously)
* **Kanban app** → `frontend/` (The Vite React application directory containing the working Kanban board)
* **Live URL** → `README.md` (See README.md for the live deployed URL of the Kanban app)
* **Slack proof** → `screenshots/` (Contains visual proof mapping the Slack workflow to screenshots)
